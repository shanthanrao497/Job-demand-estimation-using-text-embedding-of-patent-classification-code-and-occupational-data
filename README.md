# Job-demand-estimation-using-text-embedding-of-patent-classification-code-and-occupational-data

**Description:**
AI-powered web application that predicts future job demand by analyzing patent documents and occupational datasets using text embedding techniques. The system identifies relationships between emerging technologies and job roles, helping users understand future employment trends and workforce requirements.

Technologies:
Java (JSP/J2EE), MySQL, HTML, CSS, JavaScript, NLP, Text Embeddings, Apache Tomcat

Features:

* Patent data analysis
* Job demand prediction
* Occupational trend analysis
* Experience level prediction
* Job category classification
* Company location analysis
* Work setting analysis
* User authentication system
* Dataset management
* Interactive web dashboard

System Workflow:

1. User uploads or selects a dataset.
2. Patent and job-related data are processed.
3. Text embeddings are generated from patent descriptions.
4. Similarity analysis is performed between patents and job roles.
5. Demand estimation models analyze trends.
6. Job demand predictions are generated.
7. Results are displayed through the web interface.

Benefits:

* Identifies future job opportunities
* Supports workforce planning
* Helps organizations understand technology trends
* Assists job seekers in career planning
* Enables data-driven decision-making

How to Run:

1. Import the project into Eclipse/NetBeans.
2. Configure the MySQL database using the provided SQL file.
3. Deploy the project on Apache Tomcat Server.
4. Start the server.
5. Open the application in a web browser.
6. Login and access the prediction modules.

Results:

* Accurate job demand estimation
* Technology trend identification
* Occupational growth prediction
* Data-driven workforce insights
* Efficient analysis of patent-based innovations
